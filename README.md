# MadokaLiteBlog

## 简介

MadokaLiteBlog 是一个基于 .NET 8 的博客系统，使用 PostgreSQL 作为数据库，Dapper 作为 ORM 框架，NLog 作为日志框架。

## 技术栈

- .NET 8
- PostgreSQL
- Dapper
- NLog
